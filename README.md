# TileLauncherGUI

此工具专门为Win11设计，适合想在Win11上使用磁贴功能的用户，需要搭配 **[ExplorerPatcher](https://github.com/valinet/ExplorerPatcher)**使用的工具。

<aside>

💡Win10也可以使用，不过并不推荐，因为Win有类似的工具并且表现要比本工具好不少，也不需要搭配ExplorerPatcher使用

</aside>

## 使用前提：

打开Windows设置中的开发人员模式，并确保电脑已经安装了Net8.0

## 支持的功能？

- 单应用模式

        创建单个磁贴来打开或运行任意网页、应用程序、快捷方式、bat命令等

        支持快捷选择想打开的文件

        支持动态轮播功能

- 多应用模式

       支持导入你喜欢的图片

       对导入的图片根据选择的磁贴大小进行自定义切割

       一键创建磁贴

<aside>

💡在该模式下按住鼠标右键缩放滚轮可对导入的图片进行位置、大小调节

</aside>

> 该模式下不支持自定义打开程序，仅仅是一个空花瓶。或许未来会支持🧐？
> 
- 磁贴管理

        对已创建的磁贴进行可视化删除管理

> 磁贴默认储存在目录下的MyGameLaunchers文件夹内
> 
- 更新检测

       每次打开软件都会进行一次联网检测更新，若有必要更新会跳转到对应下载页面

> 感觉有点多此一举，未来可能会删除该功能，若有能力且不喜欢可以自行删除UpdateManager.cs并修改相应区域进行编译
>
